# Auto-Teleflow - Assistant

This directory contains the placeholder and architecture plan for the **Assistant** application, a planned Customer Support chatbot.

## Scope & Purpose
The assistant bot is designed to handle incoming user inquiries, provide pre-sales answers, route chats to human representatives (e.g., via WhatsApp), and integrate with existing auto-order bots.

For detailed roadmap, implementation specifications, and design plans, see [CS Assistant Plan](docs/plan.md).
